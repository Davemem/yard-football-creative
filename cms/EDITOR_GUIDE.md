# CMS/EDITOR_GUIDE.md

This guide keeps the Phase 6 Sanity editing flow simple for non-technical use.
It covers only the two content types approved for v1:

- `Program`
- `Event`

## Editing Rule

- Only publish content that is ready to appear on the public website.
- Keep each entry short enough to fit the listing cards cleanly.
- Use the approved contact or registration URL for the primary CTA.

## `Program` Entry Checklist

1. Add the `Title`.
2. Choose the `Category`.
3. Choose the `Pathway`.
4. Add a short `Status Label`.
5. Add the `Summary`.
6. Add the short `Audience` label.
7. Add the `Location`.
8. Add the `Start Date`.
9. Add the `Primary CTA Label`.
10. Add the `Primary CTA URL`.

Optional:

- `End Date`
- `Display Order`

## `Event` Entry Checklist

1. Add the `Title`.
2. Choose the `Category`.
3. Add a short `Status Label`.
4. Add the `Summary`.
5. Add the `Location`.
6. Add the `Start Date`.
7. Add the `Primary CTA Label`.
8. Add the `Primary CTA URL`.

Optional:

- `End Date`
- `Time Label`
- `Display Order`

## What Appears On The Website

Each published entry feeds one listing card with:

- category
- status
- title
- summary
- short meta details
- one primary call to action

## Keep It Simple

- Use plain language.
- Keep summaries concise.
- Avoid adding internal notes into public fields.
- If a detail is not ready to publish, leave optional fields empty rather than guessing.
